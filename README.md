# SimGarage

## Descripción

E-Commerce de productos de simracing (en español, automovilismo virtual) creado con React donde el usuario podrá navegar a través del catálogo de productos publicado, filtrar los mismos por categoría, agregar los productos deseados a un carrito de compras y luego finalizar su compra completando previamente un formulario con su información personal.

### Base de datos

Para el presente proyecto se ha utilizado Firebase Database como base de datos, donde se hace uso de las siguientes colecciones:


* Stock: Se crea un documento por cada producto, donde contiene información tal como su nombre, precio, categoría, stock, imagen (almacenada en Firebase Storage) y descripción. Es utilizada para la publicación de los productos en el sitio web.
* Orders: Se crea un documento por cada pedido de compra con la información del comprador y los productos comprados. El nombre del documento se utiliza como el número de orden.

### Librerías externas utilizadas

* React Bootstrap
* SweetAlert
